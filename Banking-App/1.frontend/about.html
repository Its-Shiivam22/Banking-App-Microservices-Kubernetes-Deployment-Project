About This DevOps Project

A real-world style Kubernetes microservices project for DevOps portfolio.

Project Overview

This project demonstrates deployment of a banking microservices application on AWS Kubernetes using kOps. The application uses a multi-page frontend, multiple backend APIs, and MySQL database connectivity.

Containerization

Each service has its own Dockerfile and image.

Kubernetes

Deployments, Services, Ingress, ConfigMap, Secret, PVC, RBAC, and HPA.

CI/CD

Jenkins builds, pushes, and deploys images to Kubernetes.

Monitoring

Prometheus and Grafana monitor pods, nodes, and namespace resources.
